# Packaging in Docker Compose
## Requirements
- docker
- docker compose

## Getting Started
### Installation
1. Clone the repo
```sh 
git clone https://github.com/alexandertolchinsky/devops-for-programmers-project-lvl1.git
```
2. Install package locally
```sh
make install
```
## How to run 
```sh 
make start
```
